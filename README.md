# freema-kaikei

メルカリの販売履歴を、freee / マネーフォワードクラウドへ取り込める会計CSVに変換するChrome拡張機能。

---

## 概要

メルカリの販売履歴ページを自動で全ページ読み込み、1取引を複合仕訳（売上高・支払手数料・荷造運賃）としてCSV出力します。個人事業主・課税事業者向けの設計です。

---

## 機能

- メルカリ販売履歴の自動全件取得（ページネーション対応）
- 月単位のフィルタリング・チェック式の個別選択
- freee 取引インポートCSV出力（複合仕訳・UTF-8 BOM付き）
- マネーフォワードクラウド 仕訳CSVの出力
- 勘定科目のカスタマイズ（設定画面）
- 選択SaaSの記憶（次回起動時に復元）

## インストール

1. このリポジトリをクローンまたはZIPでダウンロード
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をオン
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択

---

## 使い方

1. メルカリにログインした状態でChrome拡張機能アイコンをクリック
2. 出力先SaaS（freee / マネーフォワード）を選択
3. 対象年を選んで「読み込む」をクリック
4. 取得完了後、月フィルターで対象月を選択
5. 取引にチェックが入っていることを確認して「CSVエクスポート」

> **注意**: 読み込み時に背景タブが一時的に開閉します（メルカリのJS描画対応のため）。

---

## 仕訳ルール

1取引につき最大4行の複合仕訳を出力します。

| 行 | 科目 | 金額 | 税区分 |
|---|---|---|---|
| 1（収入） | 売上高 | 商品価格（税込） | 課税売上10% |
| 2 | 支払手数料 | 販売手数料（税込）× −1 | 課対仕入10% |
| 3 | 荷造運賃 | 送料（税込）× −1 | 課対仕入10% |
| 4 | 広告宣伝費 | 他費用（税込）× −1 | 課対仕入10% |

- 行2〜4は `収支区分` を空白にすることでfreeeが複合仕訳として認識
- 税額は `floor(税込金額 ÷ 11)` で計算（内税）
- 他費用が `---`（0円）の場合は行4を出力しない

---

## 設定

拡張機能アイコンの設定ボタン（⚙）から勘定科目を変更できます。

| 項目 | デフォルト |
|---|---|
| 売上勘定科目 | 売上高 |
| 未収金勘定科目 | 売掛金 |
| 手数料勘定科目 | 支払手数料 |
| 送料勘定科目 | 荷造運賃 |
| 他費用勘定科目 | 広告宣伝費 |

設定は `chrome.storage.sync` に保存されます。
